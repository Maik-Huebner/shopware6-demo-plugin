# Shopware 6 Demo Plugin

Example plugin demonstrating a clean Shopware 6 plugin structure.

## Features

- Event Subscriber example
- Service class structure
- Example API integration service
- Clean and readable plugin architecture

## Tech Stack

- PHP 8
- Symfony
- Shopware 6
- Composer

## Purpose

This repository demonstrates how I structure Shopware plugins when working with agency teams.

Focus:

- clean architecture
- maintainable code
- understandable structure for teams
